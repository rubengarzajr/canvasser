Create an interaction
In this tutorial, we will create two buttons that will show and hide an object.
Step 1: Create Images:
Click on Add in the Images window.
A new Image will be created.
Click on the new Image and then view it's Properties window.
Change the id to show-button.
Change the path to sample_images.
Change the url to sample_show.png.
Your properties should look like this:
Repeat these steps for sample_hide.png and sphere.png.
Step 2: Create Objects:
Click on Add in the Objects window.
A new Object will be created.
Click on the new Object and then view it's Properties window.
Change the id to sphere.
Change the image to sphere.
Congratulations, you have your first object showing up in Canvasser!
Create a new object and change the id to hide.
Change the image to hide-button.
Change the position to x:100 y:200.
Check the interactive checkbox so that the button is clickable.
Click on Add Action button.
Cleardown means a click will only register once. Most of the time you will have this as your first action.
Click on Add Action button and select vis.
Change id to sphere.
Change filter to object.
Try clicking on the Hide button. This sphere should vanish! Try creating a show button to make the sphere reappear.
