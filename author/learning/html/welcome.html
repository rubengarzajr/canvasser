Welcome to Canvasser!
Canvasser is a building tool that allows you to design interactive areas on a web page that work on a computer or tablet. Canvasser started as a simple wrapper on top of HTML5 canvas (which is why it's called Canvasser) but has since evolved into a more robust animation/interactive editor.
You can start working now or:
  • click on Contents in the menu above to view the manual
  • expand the Samples window to view some example interactives
