Create an animation
In this tutorial, we will create an animation that plays automatically.
Step 1: Create Image:
Click on Add in the Images window.
A new Image will be created.
Click on the new Image and then view it's Properties window.
Change the id to cube.
Change the path to sample_images.
Change the url to cube_spinner.png.
Check atlas.
Change cell width and height to 100.
Your properties should look like this:
Step 2: Create Object:
Click on Add in the Objects window.
A new Object will be created.
Click on the new Object and then view it's Properties window.
Change the id to cube.
Change the image to cube.
Step 3: Create Animation:
Click on Add in the Animation window.
A new Animation will be created.
Click on the new Animation.
View it's Properties window.
Change the id to cubemove.
Check autostart.
Change the length to 3000.
Length is measured in milliseconds.
